# ROS-Auto-Setup-Script
For Ubuntu Focal and Bionic only(for now)
Clone the repo
```Shell
git clone https://github.com/PolyU-Robocon/ROS-Auto-Setup-Script.git
```
Run the script
```Shell
cd ROS-Auto-Setup-Script
bash setupROS.bash
```
